# 📝 Azure AI Notes

This document contains concise notes and important concepts related to **Microsoft Azure AI Services**. It serves as a quick revision guide covering Azure AI fundamentals, responsible AI, machine learning concepts, Azure AI services, and generative AI.

---

## 📌 What is Artificial Intelligence?

Artificial Intelligence (AI) is the ability of machines to perform tasks that normally require human intelligence.

Examples include:

- Image Recognition
- Speech Recognition
- Language Translation
- Chatbots
- Recommendation Systems
- Decision Making

---

## 📌 Types of AI

**Narrow AI (Weak AI)**

Designed for a specific task.

Examples:

- Siri
- Alexa
- Microsoft Copilot
- Face Unlock

##

**General AI (Strong AI)**

Can perform any intellectual task similar to humans.

Currently does **not exist**.

##

**Generative AI**

Creates new content instead of only analyzing existing data.

Examples:

- ChatGPT
- Microsoft Copilot
- DALL·E
- GitHub Copilot

---

## 📌 Machine Learning

Machine Learning (ML) is a subset of AI where systems learn patterns from data without being explicitly programmed.

Workflow:

```
Data

↓

Training

↓

Machine Learning Model

↓

Prediction
```

---

## 📌 Types of Machine Learning

**Supervised Learning**

Uses labeled datasets.

Examples:

- House Price Prediction
- Email Spam Detection

##

**Unsupervised Learning**

Uses unlabeled datasets.

Examples:

- Customer Segmentation
- Clustering

##

**Reinforcement Learning**

The model learns by interacting with an environment using rewards and penalties.

Examples:

- Robotics
- Self-driving Cars
- Game AI

---

## 📌 Deep Learning

Deep Learning is a subset of Machine Learning that uses Artificial Neural Networks with multiple hidden layers.

Applications:

- Computer Vision
- NLP
- Speech Recognition
- Autonomous Vehicles

---

## 📌 Azure AI

Azure AI is Microsoft's cloud platform for building, deploying, and managing Artificial Intelligence solutions.

Azure provides pre-built APIs, machine learning tools, and AI services for developers.

---

## 📌 Azure AI Services

Azure AI Services provide ready-to-use APIs without requiring deep ML expertise.

Common services include:

- Vision AI
- Speech AI
- Language AI
- Translator
- Document Intelligence
- Azure OpenAI Service

---

## 📌 Azure OpenAI Service

Azure OpenAI provides secure access to OpenAI models through Microsoft Azure.

Supported models include:

- GPT Models
- DALL·E
- Embeddings
- Whisper

Common use cases:

- Chatbots
- Code Generation
- Document Summarization
- Content Creation

---

## 📌 Natural Language Processing (NLP)

NLP enables computers to understand and process human language.

Applications:

- Chatbots
- Sentiment Analysis
- Translation
- Text Classification
- Question Answering

---

## 📌 Computer Vision

Computer Vision enables machines to understand images and videos.

Applications:

- Object Detection
- OCR
- Face Detection
- Image Classification
- Medical Imaging

---

## 📌 Speech AI

Speech AI converts speech into text and text into speech.

Applications:

- Voice Assistants
- Live Captions
- Voice Commands
- Accessibility

---

## 📌 Responsible AI

Microsoft follows Responsible AI principles to ensure AI systems are trustworthy and ethical.

Core Principles:

- Fairness
- Reliability & Safety
- Privacy & Security
- Inclusiveness
- Transparency
- Accountability

---

```
Collect Data

Train Model

Monitor & Improve
```

## 📌 Azure AI Studio

Azure AI Studio is Microsoft's unified platform for building, testing, and deploying AI applications.

Features:

- Prompt Engineering
- Model Deployment
- Evaluation
- Safety Filters
- Playground
- Fine-tuning

---

## 📌 Prompt Engineering

Prompt Engineering is the process of designing effective prompts to improve AI responses.

Good prompts should be:

- Clear
- Specific
- Context-rich
- Goal-oriented

## 📌 Advantages of Azure AI

- Cloud-based
- Scalable
- Secure
- Easy API integration
- Enterprise-ready
- Supports Responsible AI

## 📌 Limitations

- Requires quality data
- Cloud costs
- Model bias
- Hallucinations
- Privacy considerations

## 📌 Key Azure Terms

| Term | Meaning |
|------|---------|
| Model | AI system trained on data |
| Dataset | Collection of training data |
| Prompt | Input given to an AI model |
| Token | Small unit of text processed by AI |
| Endpoint | URL used to access Azure services |
| Resource | Azure service instance |

---

## 💡 Key Takeaways

- Azure AI provides enterprise-ready AI services on the cloud.
- Machine Learning enables systems to learn from data.
- Azure AI Services simplify AI development using pre-built APIs.
- Azure OpenAI combines Microsoft's cloud security with OpenAI models.
- Responsible AI ensures ethical and trustworthy AI solutions.
- Prompt engineering plays a critical role in improving AI-generated responses.
